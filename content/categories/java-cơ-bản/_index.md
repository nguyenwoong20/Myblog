# Java Cơ Bản

Khám phá thế giới lập trình Java từ những kiến thức nền tảng cơ bản nhất.

## Bạn sẽ học được:
- Cú pháp Java cơ bản
- Biến, kiểu dữ liệu và toán tử
- Cấu trúc điều khiển (if/else, loops)
- Classes và Objects
- String manipulation
- Arrays và Collections cơ bản
- Exception handling cơ bản
